أنشئ لي موقع ويب كامل احترافي باسم "الموحد" (almowahid) باستخدام ملف واحد index.html فقط (HTML + Tailwind CSS via CDN + JavaScript نقي). الموقع يجب أن يكون متجاوب 100% مع جميع الأجهزة (mobile-first) ويدعم الاتجاه RTL بالكامل.

### التصميم العام:
- استخدم Tailwind CSS عبر الـCDN: <script src="https://cdn.tailwindcss.com"></script>
- لون الشريط العلوي: أخضر إسلامي غامق (#166534) أو emerald-700.
- الخطوط: استخدم خط "Amiri" أو "Noto Sans Arabic" عبر Google Fonts.
- الاتجاه: dir="rtl" + lang="ar".

### الهيدر (الشريط العلوي الأخضر):
- شريط علوي ثابت (sticky top-0) بلون أخضر.
- في الوسط (text-center):
  - عنوان رئيسي كبير: "أداة بحث" (لون أبيض، font-bold، text-3xl)
  - تحتها عنوان فرعي بلون باهت (text-emerald-100 opacity-75): "أداة للبحث في بعض مواقع مشايخ أهل السنة والجماعة"
- في أعلى يمين الشريط: أيقونة همبرغر (ثلاث خطوط أفقية بيضاء) تفتح قائمة جانبية من اليمين (slide-in).

### القائمة الجانبية (عند الضغط على الهامبرغر):
- تظهر sidebar من اليمين (w-80 bg-white shadow-2xl) مع زر إغلاق.
- في أعلى يسار القائمة (بسبب RTL): قسم "مشاركة الموقع" يحتوي على:
  - زر "نسخ الرابط" (يستخدم navigator.clipboard)
  - QR Code للرابط الحالي (استخدم مكتبة qrcode.js أو API https://api.qrserver.com/v1/create-qr-code/?size=200x200&data= + window.location.href)
  - العنوان فوق الـQR: "شارك الموقع"

### خانة البحث (تحت الهيدر مباشرة):
- input كبير وواضح (bg-white border-2 border-emerald-600 focus:border-emerald-700) مع placeholder: "اكتب كلمة البحث..."
- بجانبه زر "بحث" أخضر كبير (bg-emerald-600 hover:bg-emerald-700).
- أسفل الـinput أو بجانبه: قائمة منسدلة بعنوان "مكان البحث" (select أو custom dropdown جميل).

### خيارات "مكان البحث" بالضبط:
1. كل المواقع
2. موقع الشيخ فركوس حفظه الله
3. موقع الشيخ عبد الرزاق البدر حفظه الله
4. موقع الشيخ رسلان حفظه الله
5. موقع الشيخ الفوزان حفظه الله لهيئة كبار العلماء
6. موقع الشيخ إبن باز رحمه الله
7. موقع الشيخ إبن عثيمين رحمه الله لهيئة كبار العلماء
8. موقع الشيخ إبن عثيمين رحمه الله
9. موقع الشيخ ربيع بن هادي المدخلي رحمه الله
10. موقع الشيخ سليمان الرحيلي حفظه الله

### وظيفة البحث:
- عند الضغط على "بحث" أو Enter:
  - أظهر قسم "نتائج البحث" (مع loading spinner أخضر).
  - إذا اختار "كل المواقع" → يبحث في الـ9 مواقع.
  - إذا اختار موقع واحد → يبحث فقط فيه.
- تنفيذ عملي: 
  - افتح تبويب جديد لكل موقع مختار مع إضافة الكلمة في نهاية الرابط (مثل: site.com/?s=الكلمة أو /search?q=الكلمة).
  - أنشئ قسم نتائج جميل يحتوي على كروت (cards) لكل موقع: اسم الشيخ + رابط مباشر + "نتائج وهمية" (للعرض) مع إمكانية استبدالها لاحقًا بـ scraping حقيقي.
  - أضف رسالة صغيرة في الأسفل: "في الإصدار الكامل يمكن إضافة backend بـ Flask للـ scraping التلقائي".
